¿Qué es C++?

C++ es un lenguaje de programación potente y eficiente, perfecto para crear desde aplicaciones simples hasta sistemas complejos.

Estructura básica

1. Primer programa: "Hola Mundo"

Objetivo: Imprime el mensaje "¡Hola, mundo!" en la consola.

Paso 1: Incluye la librería #include <iostream>

Paso 2: Declara el uso de using namespace std;

Paso 3: Define la función int main()

Paso 4: Dentro de main(), usa cout para imprimir el mensaje. 

#include <iostream>
using namespace std;

int main() {
  cout << "¡Hola, mundo!" << endl;
  return 0;
}
